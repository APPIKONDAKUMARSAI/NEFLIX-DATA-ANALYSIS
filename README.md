# NEFLIX-DATA-ANALYSIS
# 🎬 Netflix Data Analysis using Python

## Project Overview

This project performs **Exploratory Data Analysis (EDA)** on the Netflix dataset using Python to identify trends in content distribution, genres, countries, ratings, release years, and movie durations.

The objective is to transform raw data into meaningful insights through data cleaning, analysis, and visualization.

---

## Objectives

* Analyze Movies vs TV Shows
* Explore yearly content trends
* Identify popular genres
* Analyze country-wise content production
* Study movie duration and TV show seasons
* Understand audience ratings

---

## Technologies Used

* Python
* Jupyter Notebook
* Pandas

* Matplotlib
* Seaborn

## Project Workflow

## Key Findings

### 🎥 Content Distribution

* Movies account for approximately **70%** of Netflix's library.
* TV Shows represent around **30%**.

### 📈 Content Trends (2008–2021)

* Movie releases increased significantly from **2016**.
* Peak content additions occurred in **2019**.
* Decline observed during **2020–2021**.

### 📺 TV Show Trends

* TV Show releases peaked during **2019–2020**.
* Declined in **2021**.

### 🎭 Top Genres

* International Movies
* Dramas
* Comedies
* International TV Shows
* Documentaries

### ⏱️ Movie Duration

* Most movies run between **80–120 minutes**.

### 📺 TV Show Seasons

* Most TV Shows have only **one season**, indicating a preference for limited-series content.

### 🌍 Top Content Contributors

| Country       | Titles |
| ------------- | -----: |
| United States |  4,509 |
| India         |  1,046 |
| Canada        |    803 |

### 👥 Target Audience

* Most content targets viewers aged **15+ and 18+**.

### 📌 Runtime Outliers

* Dramas and International Movies contain the largest runtime variations.
